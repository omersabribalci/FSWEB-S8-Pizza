# Pizza Order Website

A React-based pizza ordering application built as part of the Workintech Web Development training program.

## Features

- Interactive pizza ordering form with customization options
- Order form with validation and error handling
- Responsive design for mobile and desktop devices
- Real API integration using Axios (Mock API: reqres.in)
- Page navigation with React Router
- Order confirmation page
- Semantic HTML structure
- E2E testing with Cypress

## Tech Stack

- **Frontend**: React 18+
- **HTTP Client**: Axios
- **Routing**: React Router v6
- **Testing**: Cypress
